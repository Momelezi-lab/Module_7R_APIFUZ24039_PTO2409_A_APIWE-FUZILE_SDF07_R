Black Jack

Black Jack App

I developed a Blackjack game using HTML, CSS, and JavaScript to create an engaging and interactive experience. The game simulates the classic card game, allowing players to draw cards, calculate sums, and aim for a total of 21 without exceeding it. The interface features a clean design with a casino-inspired theme, incorporating vibrant colors and intuitive buttons for starting a game or drawing new cards. JavaScript powers the game's logic, dynamically updating the player's card total and determining the outcome of each round. CSS styling was applied to create a visually appealing layout, while responsive design ensures a smooth experience across different devices. This project highlights my ability to combine functionality with aesthetics, showcasing skills in interactive front-end development.
